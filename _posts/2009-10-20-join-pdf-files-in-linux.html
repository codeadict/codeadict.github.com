--- tags: - GNU/Linux cli type: post status: publish title: Join PDF files in linux meta: _edit_last: "7940715" published: true layout: post --- Hello linux users: in this post i will show how to join two or more PDF files into one. You have to install the Ghostscript driver to do that, typing in a terminal(is for Debian and Ubuntu): {% highlight bash %} apt-get install gs apt-get install pdftk {% endhighlight %} Now we join the PDFs with this command: {% highlight bash %} gs -dBATCH -dNOPAUSE -q -sDEVICE=pdfwrite -sOutputFile=unidos.pdf  file1.pdf  file2.pdf {% endhighlight %} Here is the explanation of command: gs: Call to the Ghostscript program. -dBATCH: Close the Ghostscript when terminate the process. -dNOPAUSE: Dont pause the program to aks for user interaction. -q: Dont show messages during the process. -sDEVICE=pdfwrite: Use Ghostscript PDF generator to do the process. -sOutputFile=unidos.pdf: The name of the result file. You can also join all PDF files in a directory typing *.pdf.
